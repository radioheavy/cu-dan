## Kurulum

Projeyi yerel makinenizde çalıştırmak için aşağıdaki adımları izleyin:

1. Repoyu klonlayın:
   ```
   git clone https://github.com/kullaniciadi/proje-adi.git
   ```

2. Proje dizinine gidin:
   ```
   cd proje-adi
   ```

3. Gerekli bağımlılıkları yükleyin:
   ```
   npm install
   ```

4. Projeyi çalıştırın:
   ```
   npm start
   ```
